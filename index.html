1. 9장 포인터

포인터는 메모리의 주소를 가지고 있는 변수

표현방법

int *p; //포인터 p선언 (간접 참조 연산)

p = &number; //변수 number의 주소가 포인터 p로 대입



포인터 타입 ++연산후 증가하는 값
char 1
short 2
int 4
float 4
double 8


표시 내용
*p++ p를 가르키는 내용을 가져오고 p를 증가.
(*p)++ p가 가르는 위치 값 증가.
v = * ++p p를 증가시킨 후 p가 가르키는 값을 v에 대입.
v = ++*p p를 가르키는 값을 가져오고 그 값을 증가 시켜 v에 대입.



값에 의한 호출

함수가 호출될 때 복사본이 함수로 전달되면 값에 의한 호출이다. 함수 안에서 매개 변수를 변경하여도 원본에는 영향을 주지 않는다.


참조에 의한 호출

만약 함수가 호출될 때 원본을 함수로 전달하는 방법이다. 함수 안에서 매개 변수를 변경하면 원본 변수가 변경된다.

2. 10장 문자열

문자 입출력

함수 설명
getchar() 하나의 문자를 받아서 반환한다. 버퍼를 사용한다.
putchar() 하나의 문자를 받아서 출력한다.
getch() 하나의 문자를 받아서 반환한다. 버퍼를 사용하지 않는다.
putch() 하나의 문자를 받아서 출력한다.

문자열 입출력

입출력 함수 설명
scanf("%s", s) 문자열을 읽어서 배열 s[]에 저장.
printf("%s", s) 배열 s[]에 저장되어 있는 문자열을 출력.
get_s(char s[], int length) 한 줄의 문자열을 읽어서 배열 s[]에 저장.
puts(char s[]) 배열 s[]에 저장되어 있는 한 줄의 문자열을 출력.

문자열 입출력

입출력 함수 설명
strlen(s) 문자열 s의 길이를 구한다.
strcpy(s1, s2) s2를 s1에 복사한다.
strcat(s1, s2) s2를 s1의 끝에 붙여넣는다.
strcmp(s1, s2) s1과 s2를 비교한다.
strcpy(s1, s2, n) s2의 최대 n개의 문자를 s1에 복사한다.
strcat(s1, s2, n) s2의 최대 n개의 문자를 s1의 끝에 붙여넣는다.
strcmp(s1, s2, n) 최대 n개의 문자까지 s1과 s2를 비교한다.
strchr(s, c) 문자열 s안에서 문자 c를 찾는다.
strstr(s1, s2) 문자열 s1에서 문자열 s2를 찾는다.
